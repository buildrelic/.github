# Relic

### The _source of truth_ for AI-native companies.

> **Stop guessing. Start _remembering._**

Relic ingests artifacts from every tool your team uses — docs, code, threads, tickets, meetings — and ranks them into a single, queryable **memory graph** that both humans and AI agents can trust.

---

## Why Relic

Software is moving from _people-driven_ to _agent-assisted_. The bottleneck is no longer how fast we can write code. It's how fast we can **agree on what's true** — and trust that agents will build to that truth.

Without a memory layer, every conversation, every spec, every meeting evaporates. Vector search returns the gist. Embeddings forget what overrides what. **Hallucination is the default.**

Relic fixes that.

## What it does

- **🧠 Memory Graph** — Ingests and ranks artifacts from across your stack into one unified source of truth.
- **🔎 Unified Query Layer** — One place to ask "what's true," accessible to both humans and AI agents.
- **🔌 Multi-source Integration** — Connects to the tools your team already lives in.
- **🛡️ Enterprise-ready** — SOC 2, GDPR, and HIPAA compliant from day one.
